# 应用图标生成提示词

## 文件信息
- **文件名**: icon.png
- **尺寸**: 512 x 512

## Prompt (English)
```
Mobile game app icon for a Dr. Mario style puzzle game. 
Cute cartoon capsule character (half red, half blue) as the main element. 
Clean white or light blue background. 
Rounded square icon format. 
Eye-catching, colorful, professional. 
Suitable for WeChat Mini Game store.
```

## 中文描述
马里奥医生风格益智游戏的移动应用图标。可爱的卡通胶囊角色（一半红，一半蓝）作为主要元素。干净的白色或浅蓝色背景。圆角方形图标格式。醒目、多彩、专业。适合微信小游戏商店。
