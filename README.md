# 双色胶囊 / Dual Capsule

> FC《马里奥医生》复刻版 | Cocos Creator 3.8.8 | 微信小游戏

## 游戏简介

《双色胶囊》是经典 FC 游戏《马里奥医生》的精神续作。玩家扮演药剂师，通过旋转和投放双色胶囊，使 4 个或以上同色方块连成一线进行消除，目标是清空瓶内所有病毒。

## 核心玩法

- 🎮 控制双色胶囊的移动和旋转
- 🧬 使 4 个以上同色方块连成一线消除
- ⚡ 连锁消除获得高分
- 🏆 清空所有病毒即可过关
